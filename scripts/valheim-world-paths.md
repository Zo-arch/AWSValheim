# World Files

Valheim world files are usually a pair with the same basename:

- `MyWorld.db`
- `MyWorld.fwl`

Common local paths on Windows:

- `%USERPROFILE%\AppData\LocalLow\IronGate\Valheim\worlds_local`
- `%USERPROFILE%\AppData\LocalLow\IronGate\Valheim\worlds`

Server target path in this setup:

- `/srv/valheim/worlds_local`
